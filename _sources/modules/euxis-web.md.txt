# euxis-web

`euxis-web` provides JavaScript/TypeScript crypto package outputs and interoperability layers.

## Focus Areas

- Efficient cryptographic operations with predictable resource usage
- Secure defaults for package supply-chain and dependency posture
- Portability across Node.js and browser-adjacent integration environments

## Quality Controls

- Package manifest: `euxis-web/package.json`
- Stable package gate: `euxis-ops/quality/run_crypto_packages_tests_stable.sh`
- Package policy: `euxis-ops/perf/package_resource_policy.json`
- Workspace excellence gate: `euxis-ops/quality/validate_package_excellence.py`
