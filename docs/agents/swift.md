# Swift Agents & Playbooks

The Euxis Swift playbook integrates directly with SPM (Swift Package Manager) to enforce iOS and macOS strictness.

## Supported Tools

*   **Swift Package Manager** (SPM).
*   **XCTest**: Swift testing and mocking.
*   **SwiftLint**: Strict code styling.

## Usage

```bash
euxis playbook run swift "Extract the networking layer into a separate SPM module"
```
