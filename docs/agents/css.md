# CSS & HTML Agents & Playbooks

Euxis understands modern web layout engines, CSS grid mechanics, and component isolation.

## Supported Tools

*   **TailwindCSS** (or standard PostCSS).
*   **Stylelint**: CSS linting.
*   **HTMLHint**: HTML semantic checking.

## Usage

```bash
euxis playbook run css "Create a responsive, accessible navigation bar"
```
