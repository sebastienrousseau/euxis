# JavaScript & TypeScript Agents & Playbooks

The Euxis JS/TS playbook is geared towards modern web environments, emphasizing strict typing and performance.

## Supported Tools

The playbook natively understands:
*   **TypeScript Compiler (tsc)**: Strict type checking.
*   **ESLint / Prettier**: Linter and formatter.
*   **Vitest / Jest**: Test runners.
*   **Node/Bun/Deno**: Runtime selection and handling.

## Usage

```bash
euxis playbook run javascript "Implement a robust debounce utility function"
```
