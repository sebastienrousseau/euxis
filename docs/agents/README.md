# Euxis Agent Ecosystem

Euxis features 53 specialized agents designed for comprehensive AI engineering tasks. 

## Core Agents

Core agents handle the high-level orchestration, planning, and review of your project.

*   **architect**: Designs systems and creates execution manifests.
*   **orchestrator**: Coordinates multi-agent workflows.
*   **auditor**: Reviews code quality and compliance.
*   **guard**: Security gatekeeper and validator.
*   **planner**: Creates project plans and roadmaps.
*   **reviewer**: Performs rigorous code review and approval.

## Fleet Agents

Fleet agents are specialists that perform specific tasks like coding, testing, designing, and researching.

*   **Development**: debugger, tester, repairer, optimizer, maintainer.
*   **Research**: researcher, deep-researcher, investigator.
*   **Security**: pentester, sentinel, watchdog, gatekeeper.
*   **Creative**: designer, writer, animator.

## Integration Examples

**Single Agent Execution**
```bash
euxis architect "Design a REST API for user auth"
```

**Squad Deployment**
```bash
euxis squad deploy quality "Audit the payment module"
```

**Playbook Execution**
```bash
euxis playbook run rust "Implement standard Axum server"
```
