Euxis Documentation
====================

Welcome to the Euxis Fleet documentation. Euxis is a multi-provider AI agent
framework that coordinates specialist agents across multiple AI providers for
engineering tasks.

Current Quality Snapshot (February 18, 2026)
--------------------------------------------

- Provider shell suite is stable: 53/53 passing
- Provider function-level shell coverage is at 100% (63/63)
- Additional shell suites are still being hardened for full green CI
