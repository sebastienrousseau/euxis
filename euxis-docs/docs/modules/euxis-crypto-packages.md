# euxis-crypto-packages

`euxis-crypto-packages` provides JavaScript/TypeScript crypto package outputs and interoperability layers.

## Focus Areas

- Efficient cryptographic operations with predictable resource usage
- Secure defaults for package supply-chain and dependency posture
- Portability across Node.js and browser-adjacent integration environments

## Quality Controls

- Package manifest: `euxis-crypto-packages/package.json`
- Stable package gate: `scripts/quality/run_crypto_packages_tests_stable.sh`
- Package policy: `scripts/perf/package_resource_policy.json`
- Workspace excellence gate: `scripts/quality/validate_package_excellence.py`
