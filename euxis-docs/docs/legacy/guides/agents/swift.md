# Swift Playbook

The Euxis Swift playbook integrates natively with Swift Package Manager (SPM). Use this playbook to enforce iOS and macOS strictness throughout your application stack.

## Toolchain Integration

Euxis operates exclusively on established foundational tools.

* **Swift Package Manager (SPM):** Resolve dependencies and abstract build phases.
* **XCTest:** Execute native Swift unit testing and mocking frameworks.
* **SwiftLint:** Enforce strict code styling and conventions.

## Execute the Playbook

Delegate complex modular extraction patterns to the agent squad. 

```bash
euxis playbook run swift "Extract the networking layer into a separate SPM module."
```
